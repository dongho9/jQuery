이미지 갤러리

12장의 작품사진 이미지를 사용하여 이미지 갤러리 구현하기

구현순서

  1. 내비게이션 메뉴 5개를 만들고 12장의 이미지 목록을 가로4개 세로3줄로 배치
  2. 메뉴 클릭시 해당 주제에 맞는 이미짐나 화면에 보이도록 한다.(정렬기능)
  3. 메뉴 클릭시 해당 메뉴에 서식을 적용하여 현재 선택된 상태를 표시한다.(addClass)
  4. 갤러리 이미지에 마우스 오버 이벤트를 사용하여 캡션이 아래에서 위로 올라오도록 한다.(animate)
  5. 이미지 클릭(click)시 해당 경로의 이미지(src, attr)가 모달윈도(modal)에 뜨도록 한다.
  6. 모달윈도에서 닫기 버튼을 클릭(click)시 사라지도록(hide) 한다.
  7. 모달윈도에서 이전(prev), 다음(next) 버튼 클릭(click)시 해당 사진이 나오도록(show)한다.
  8. 모달윈도에서 이미지의 하단에 캡션(caption)내용이 표시되도록 한다.(alt속성값 출력)
  9. 모달윈도에서 이미지의 하단에 페이지 번호(page_n)가 표시되도록 한다.(1/12출력)
  10. 이미지에 마우스 오버시 다양한 효과(filter)를 적용한다.
